# Amazon Purchase Data

Amazon Purchase Data is a Python program for finding various Amazon spending stats. 

You must now request your Amazon data (https://www.amazon.com/hz/privacy-central/data-requests/preview.html)
A dummy csv file has been provided. 

## Installation

Install and run Anaconda prompt (https://www.anaconda.com/download) to use required packages. [pandas]

Drop Amazon Purhcase data .csv into root folder and rename to amazon-orders.csv

Navigate to purhcase data folder in Anaconda prompt.

Run python main.py

### Usage

Type in the corresponding number for the operation you wish to perform. 

1. Total Spent
2. Average Spent per Order
3. Total Sales Tax Spent
4. Largest Order
5. Smallest Order
